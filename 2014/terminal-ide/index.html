ZSH > all

  • cd au format court (cd ~/p/c/s => tab => cd ~/projects/chadrien/slides)
  • Complétion de presque tout
  • Correction des typos
  • RPROMPT (pour avoir un beau thème)
  • Et plein d'autres

VIM

Vi IMproved

Les plus

  • Multi plateformes (Linux, OS X, Windows)
  • Plugins (pleins)
  • Dans le terminal (si on le maîtrise en local, on le maîtrise sur un serveur distant)
  • Couvre une large quantité de languages
  • Ça impressionne quand même

Les moins

  • Courbe d'apprentissage
  • ctags parfois volumineux
  • Comparé à certains, peut manquer d'outils de refactoring (ex. intellij)
  • On passe pour un nerd

Tmux

Multiplexeur de terminaux

  • Plusieurs sessions virtuels dans une même fenêtre
  • Possibilité de détacher/rattacher les sessions
  • Layouts en "onglets" et "panneaux"

Tmuxinator

Enregistrez les configurations tmux de vos projets

git ? tig !

Un client ncurses pour git

Une configuration universelle

Tous ces logiciels sont disponibles sous Linux comme sous OS X, leurs configurations sont donc utilisables partout.

Un répo. Github et vous êtes partis (les dotfiles de Zach Holman)

Utilisable à distance

Tout ça se passe dans un terminal, il est donc tout à fait possible d'installer et configurer tout ceci sur un serveur quelque part sur les internets pour ensuite vous y connecter en SSH.

Vous devez travailler sur une machine inconnue sur laquelle rien n'est fait pour coder ? Pas grave, une connexion SSH et vous êtes lancés !
